地图离线瓦片配置说明
========================================

本目录用于存放离线地图瓦片源，支持 xyz 格式的离线图包。
系统会自动识别并加载本目录下的所有地图源。

目录结构要求：
----------------------------------------
map_tiles/
  ├── source_name1/          (源名称，例如 google_satellite)
  │     ├── 0/               (缩放级别 Zoom 0)
  │     │     ├── 0/         (X坐标)
  │     │     │     └── 0.jpg  (Y坐标.图片格式)
  │     ├── 1/               (缩放级别 Zoom 1)
  │     │     ├── 0/
  │     │     │     └── 0.jpg
  │     │     └── 1/
  │     │           └── 0.jpg
  │     └── ...
  ├── source_name2/
  │     └── ...

操作步骤：
----------------------------------------
1. 使用地图下载器（如太乐地图下载器、全能电子地图下载器等）下载地图。
2. 导出格式选择 "XYZ" 或 "标准瓦片结构"（即 z/x/y.png 或 z/x/y.jpg）。
3. 将下载后的文件夹（包含数字文件夹的那个上级目录）重命名为您想要的源名称（如 'my_city_map'）。
4. 将该文件夹放入 NAS 上的 `jeje_webos/map_tiles` 目录中。
5. 刷新网页地图，在图层切换列表中即可看到该离线源。

注意：
- 文件夹名称不要包含特殊字符。
- 支持的瓦片格式：.png, .jpg, .jpeg, .webp
- 放入即生效，无需重启服务。
