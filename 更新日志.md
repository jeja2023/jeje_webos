# 更新日志 (Changelog)

## v2.0.4 (2025-12-18) - 控制面板功能
- 🎛️ **用户管理控制面板**：
    - **快速创建用户**：管理员可直接添加用户，无需注册审核流程，支持设置角色。
    - **重置用户密码**：管理员可一键重置任意用户密码，提升运维效率。
    - **界面优化**：操作按钮统一调整，新增"添加用户"主操作按钮。
- 🔍 **全局搜索 (Spotlight)**：
    - **聚合搜索**：按下 `Ctrl + K` 或 `Ctrl + Space` 唤起搜索框，可同时搜索应用、系统设置项和云端文件。
    - **快速导航**：支持键盘光标键选择和回车打开，显著提升操作效率。
- 🔔 **通知中心增强**：
    - **通知持久化**：系统通知不再只是即时弹窗，现会保存到消息中心供随时查阅。
    - **分类图标**：消息列表根据类型（成功/警告/错误）显示不同图标。
    - **应用集成**：应用安装/卸载等操作自动记录到通知中心。
- 👤 **个人中心优化**：
    - **头像裁切**：上传头像前支持拖拽移动和缩放裁切，确保头像比例正确。
    - **圆形预览**：所见即所得的圆形取景框，模拟最终显示效果。
- 🎨 **样式优化**：
    - 系统设置按钮样式统一为全局风格。
- 🔧 **代码清理**：
    - 修复 `datetime.utcnow()` 弃用警告。
    - 修复 Pydantic V2 `class Config` 弃用警告。
    - 修复 HTTP 状态码常量弃用警告。

## v2.0.3 (2025-12-18) - 应用市场功能完善
- 🏪 **应用市场**：
    - **离线安装**：支持通过 `.jwapp` 离线包上传安装第三方模块。
    - **覆盖确认**：重复上传已存在模块时，弹窗确认是否覆盖。
    - **规范化流程**：上传后模块处于"待安装"状态，需管理员在应用市场中手动安装，再在应用管理中启用。
    - **创建应用**：支持一键创建新模块模板，自动生成后端和前端代码。
    - **删除应用**：支持删除未安装的模块代码和数据库表。
- 🔧 **组件增强**：
    - **Toast.loading**：新增加载中提示组件，支持手动关闭和内容更新。

## v2.0.2 (2025-12-17) - 快传模块上线
- ⚡ **快传功能**：
    - **跨设备传输**：支持局域网内设备间高速文件传输，无需第三方服务。
    - **传输码机制**：发送方生成 6 位数字传输码，接收方输入后即可连接。
    - **实时进度**：WebSocket 实时同步传输进度，双端状态同步。
    - **断线重连**：智能轮询机制保障弱网环境下的稳定性。
    - **历史记录**：完整的传输历史和统计数据，支持筛选和删除。
    - **自动下载**：接收方传输完成后自动触发文件下载。
- 🎨 **界面优化**：
    - **拖拽区域**：全新设计的文件上传区域，渐变图标、悬停动画。
    - **历史列表**：双行布局（文件名 + 元信息），更好的信息层次。
    - **空状态**：优化空数据提示，添加视觉图标。

## v2.0.1 (2025-12-16) - 文件管理与体验增强
- 📁 **文件管理增强**：
    - **右键菜单**：新增文件/文件夹右键菜单，支持打开、预览、移动、重命名等快捷操作。
    - **拖拽移动**：支持将文件直接拖拽到文件夹或侧边栏目录树进行移动。
    - **配额监控**：侧边栏新增存储配额进度条，实时展示空间使用情况。
    - **增强预览**：优化图片、视频、PDF、代码等多种格式的在线预览体验。
- 📝 **文档与规范**：
    - **名称统一**：将“文件管理器”统一更名为“文件管理”，将“主题美化”统一为“主题”。
    - **帮助中心**：更新文件管理帮助文档，补充新功能使用说明。
    - **系统修复**：修复右键菜单在复杂布局下的定位偏移问题。

## v2.0.0 (2025-12-15) - 统一发布版本
- ✨ **核心特性**：全新多窗口系统 (Multi-Window System)
    - **真·多任务并行**：支持同时打开多个应用窗口，状态独立互不干扰。
    - **窗口管理**：实现了原生的窗口体验，支持拖拽移动、8个方向自由缩放、最小化（隐藏至Dock）、最大化（全屏覆盖）。
    - **智能层级**：重构 Z-Index 管理，确保当前操作窗口始终置顶。
    - **关于本机**：重新设计了“关于本机”界面，更具现代感。
- 🎨 **体验优化**：
    - **Dock 栏**：修复指示器逻辑，精准显示运行中的应用。
    - **地址栏同步**：窗口焦点与浏览器 URL 实时同步，刷新网页可恢复当前窗口状态。
    - **视觉规范**：优化窗口尺寸和布局。
